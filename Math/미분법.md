## 1. 평균값 정리

- 함수 f(x)가 닫힌 구간 [a,b]에서 연속이고 열린 구간 (a,b)에서 미분 가능하면 ![img](미분법.assets/DRW00001b1c80f8.gif)  인 c가 a와 b 사이에 적어도 하나 존재한다.

-   ![img](미분법.assets/DRW00001b1c80fd.gif)  는 [a,b] 구간에서의 <u>평균변화율</u>
- f'(c)는 x=c에서의 <u>미분계수</u>



## 2. 롤 정리(Rolle's theorem)

- 함수 f(x)가 닫힌 구간 [a,b]에서 연속이고 열린 구간 (a,b)에서 미분 가능할 때, f(a)=f(b)이면 f'(c)=0 (a<c<b)인 **c**가 열린 구간 (a,b)에 적어도 하나 존재



## 3. 속도와 속력

- 거리 : 방향x, 크기o, 스칼라양

- 변위 : 위치 변화, 방향o, 크기o, 벡터양,   ![img](미분법.assets/DRW00001b1c8102.gif)  
- 속력 = 이동거리 / 시간
- 속도 = 변위 / 시간



## 4. 연쇄 법칙(chain rule)

- 두 함수를 합성한 합성함수의 미분법
-    ![img](미분법.assets/DRW00001b1c810c.gif)  
- 두 함수 f : X->Y, g : Y->Z가 주어졌을때, 집합 X의 임의의 원소 x에 대해 f(x)를 대응시키고, f(x)를 g(f(x))에 대응시켜서 X에서 Z로의 함수를 만들 수 있음.



## 5. 오차역전파

1. 계산그래프

- 장점 

  - 국소적 계산으로 각 노드의 계산에 집중하여 문제를 단순화 함.

  - 역전파로 미분을 효율적으로 계산함.

  

2. 순전파(forward propagation)와 역전파(backward propagation)

- 컴퓨터는 주어진 입력값을 신경망을 거쳐 출력값으로 반환합니다. 신경망은 <입력 - 신경망 - 출력>과 같이 좌측에서 우측으로의 진행방향을 갖습니다. 그러나 훈련을 위해서 오차 역전파(Back-propagation) 과정에서는 이와는 반대의 진행방향을 갖습니다. 오차 역전파 과정은 컴퓨터가 예측값의 정확도를 높이기 위해 출력값과 실제 예측하고자 하는 값을 비교하여 가중치를 변경하는 작업을 말합니다. ([출처](https://www.datamaker.io/posts/32/))





3. 오차역전파

- 역전파는 계산 결과와 정답의 오차를 구해서 이 오차에 관여하는 노드 값들의 가중치와 편향을 수정함.
- 이때 오차역전파(back)는 오차가 작아지는 방향으로 반복해서 수정
- 이 횟수가 커지면 그만큼 정확성은 높아지지만 시간이 오래 걸리는 단점이 있음.
- 횟수가 작아지면 정확성은 떨어지지만 시간이 단축되는 장점.
- 이 횟수의 주기를 1에포크(epoch)라 함.
- 에포크를 늘리면서 가중치와 편향을 업데이트(학습)하여 점점 오차를 줄여 나감.



4. 오차역전파 계산방식

1). 입력 값에 가중치를 곱한 값과 편향을 합하여 그 값이 임계치인 0을 넘으면 1을 출력, 그렇지 않으면 0을 출력하는 순전파 과정을 거침.

2). 출력값과 정답의 차이인 오차를 구함.

3). 역방향으로 오차를 줄이는 가중치 값으로 수정

4). 출력층의 가중치 값 수정 -> 은닉층의 가중치 값 수정



![img](../../../Data Analysis/이미지/TIL_Math_미분법 05)

[그림출처](https://thebook.io/080228/part03/ch08/01-01/)



- 가중치 수정할 때,

  - 순전파에서 계산한 y = g(f(x))의 편미분 값을 오차에 곱해서 하류 노드(은닉층)에 전달함.

  - `편미분을 사용하는 이유` : 수많은 노드에 부여되는 모든 가중치 값을 고려할 필요 없이 연결된 가중치만 고려하면 되기 때문이다.
  - 연쇄 법칙을 이용하면 출력층과 입력층 사이에 은닉층이 많아도 간단한 미분으로 기울기를 계산할 수 있기 때문이다.
