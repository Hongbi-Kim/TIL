# 산술연산메서드

- 기본 모듈

`import numpy as np`
`from pandas import Series, DataFrame`



```python
df1 = DataFrame(np.arange(1,17).reshape(4,4))
df1
#    0   1   2   3
#0   1   2   3   4
#1   5   6   7   8
#2   9  10  11  12
#3  13  14  15  16
```



- 행별(서로 다른 행끼리)

```python
df1.sum(axis=0) # 행별(서로 다른 행끼리)
#0    28
#1    32
#2    36
#3    40
#dtype: int64
```



- 컬럼별(서로 다른 열끼리)

```python
df1.sum(axis=1) 
#0    10
#1    26
#2    42
#3    58
#dtype: int64
```



```python
df1.iloc[:,0].sum()   #28
df1.iloc[:,0].mean()  #17
```



- 결측값

```python
df1.iloc[0,0] = np.nan
df1
df1.iloc[:,0].mean()
#skipna = True(default) 자동으로 NaN 무시하고 연산
```



- 평균값(최대 또는 최소) 대치

```python
df1.iloc[:,0].isnull() # 조건(boolean)
#0     True
#1    False
#2    False
#3    False
#Name: 0, dtype: bool
```

> 결측값 0으로 대치

```python
df1.iloc[:,0][df1.iloc[:,0].isnull()] = df1.iloc[:,0].mean()
df1[df1.notnull()] # 데이터프레임 전체에서 NaN 값 확인
#      0   1   2   3
#0   9.0   2   3   4
#1   5.0   6   7   8
#2   9.0  10  11  12
#3  13.0  14  15  16
```



- 여러가지 연산
  - df1.iloc[:,0].var()    #분산
  - df1.iloc[:,0].std()    #표준편차
  - df1.iloc[:,0].min()    #최소값
  - df1.iloc[:,0].max()   #최대값
  - df1.iloc[:,0].median() #중위수(중위값)



- 조건

```python
df1.iloc[:,0] >= 10
#0    False
#1    False
#2    False
#3     True
#Name: 0, dtype: bool
```

> 조건에 만족하는 개수 확인

```python
(df1.iloc[:,0] >= 10).sum() #1  
```
