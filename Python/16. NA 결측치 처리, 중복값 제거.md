# 16. NA 결측치 처리, 중복값 제거

## 1. NA (결측치) 처리

- 숫자형 NA (float type), 문자형 NA



1. NA 수정

```python
S1
# 0    1.0
# 1    2.0
# 2    3.0
# 3    NaN
# dtype: float64

S2
# 0      a
# 1      b
# 2      c
# 3    NaN
# dtype: object
```



- s1.fillna(0) :  `fillna()` 사용한 치환 -> 제일 많이 활용함
- s2.replace(np.nan,'a') : `replace` 활용, 값 치환 메서드 NA 치환 가능

- s1[s1.isnull()] = 0 : `조건 색인`  해서 NA 처리



2. NA로 수정

```python
s3
# 0    서울
# 1     .
# 2    대전
# 3    대구
# 4     .
# 5    부산
# dtype: object
```



- '.'을 NaN으로 바꿈

```python
s3_new = s3.replace('.',np.nan)
# 0     서울
# 1    NaN
# 2     대전
# 3     대구
# 4    NaN
# 5     부산
# dtype: object
```



3. NA를 이전 값/ 이후 값으로 수정

- NA를 앞에 있는 값으로 치환

```python
s3_new.fillna(method = 'ffill') 
# 0    서울
# 1    서울
# 2    대전
# 3    대구
# 4    대구
# 5    부산
# dtype: object
```

```python
s3_new.ffill()
# 0    서울
# 1    서울
# 2    대전
# 3    대구
# 4    대구
# 5    부산
# dtype: object
```



4. NA를 갖는 행, 열 제거

```python
df1
#     A    B    C     D
# 0 NaN  2.0  3.0   4.0
# 1 NaN  NaN  7.0   8.0
# 2 NaN  NaN  NaN  12.0
# 3 NaN  NaN  NaN   NaN
```



- NA를 하나라도 포함한 행 제거

```python
df1.dropna()
# Empty DataFrame
# Columns: [A, B, C, D]
# Index: []
```

```python
df1.dropna(how='any') 
```



- 모든 값이 NA인 행 제거(결측시 처리시 반드시 사용할 것)

```python
df1.dropna(how='all')
#     A    B    C     D
# 0 NaN  2.0  3.0   4.0
# 1 NaN  NaN  7.0   8.0
# 2 NaN  NaN  NaN  12.0
```



- `thresh = `

  NA 아닌 값이 최소 2개 이상이면 제거하지 않음.(실무 유용)

```python
df1.dropna(thresh=2)
    A    B    C    D
0 NaN  2.0  3.0  4.0
1 NaN  NaN  7.0  8.0
```



- 특정 컬럼이 모두 NA로만 구성되어 있으면 해당 컬럼 제거

```python
df1.dropna(axis=1, how='all')
#      B    C     D
# 0  2.0  3.0   4.0
# 1  NaN  7.0   8.0
# 2  NaN  NaN  12.0
# 3  NaN  NaN   NaN
```



- C 컬럼에 NA가 있는 행 제거 (실무 유용)

```python
df1.dropna(subset = ['C'])
#     A    B    C    D
# 0 NaN  2.0  3.0  4.0
# 1 NaN  NaN  7.0  8.0
```



## 2. 중복값 제거

1. 중복값 처리

```python
s1 = Series([1,1,2,3,4])
s1.unique() # 유일한 값 확인
# array([1, 2, 3, 4], dtype=int64)
```



2. 중복된 값 확인(boolean으로 반환)

```python
s1.duplicated() 
# 0    False
# 1     True
# 2    False
# 3    False
# 4    False
# dtype: bool
```



3. 중복값 제거

```python
s1.drop_duplicates()
# 0    1
# 2    2
# 3    3
# 4    4
# dtype: int64
```



```python
df3
#    A   B    C
# 0  1  10  100
# 1  1  10  200
# 2  3  30  300
# 3  4  40  400
```



- 모든 컬럼의 값이 일치하는 행 제거

```python
df3.drop_duplicates()
#    A   B    C
# 0  1  10  100
# 1  1  10  200
# 2  3  30  300
# 3  4  40  400
```



- A, B 컬럼 값이 일치하는 행 제거

```python
df3.drop_duplicates(subset=['A','B'])
#    A   B    C
# 0  1  10  100
# 2  3  30  300
# 3  4  40  400
```



> `keep='last'`

```python
df3.drop_duplicates(subset=['A','B'],keep='last')
#    A   B    C
# 1  1  10  200
# 2  3  30  300
# 3  4  40  400
```
